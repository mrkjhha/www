1. 모달
   
    (modal.html)
    -클릭 메뉴 부분 <button>,<a>   =>  data-toggle="modal" / data-target="#myModal" ,  href="#myModal"

      <button class="btn btn-primary btn-lg" data-toggle="modal" data-target="#myModal">
          여기를 클릭하시면 모달 윈도우가 실행됩니다.  
      </button>

      <a href="#myModal" data-toggle="modal">
          여기를 클릭하시면 모달 윈도우가 실행됩니다. 
      </a>

    -기본 구조
      -<div class="modal"><div class="modal-dialog"><div class="modal-content"> 의 구조로 되어있다.
      -<div class="modal-header"> ,  <div class="modal-body"> , <div class="modal-footer">
      -<div class="modal fade" id="myModal"  role="dialog" >
     
 <div class="modal fade" id="myModal" tabindex="-1" role="dialog" aria-labelledby="myModalLabel" aria-hidden="true">
  <div class="modal-dialog">
    <div class="modal-content">
      <div class="modal-header">
        <button type="button" class="close" data-dismiss="modal"><span aria-hidden="true">&times;</span><span class="sr-only">Close</span></button>
        <h4 class="modal-title" id="myModalLabel">모달 제목 </h4>
      </div>
      <div class="modal-body">
      <p>여기는 내용이 들어 가는 곳 </p>
      </div>
      <div class="modal-footer">
        <button type="button" class="btn btn-default" data-dismiss="modal">닫기</button>
        <button type="button" class="btn btn-primary">변경 사항 저장</button>
      </div>
    </div> <!-- 모달 콘텐츠 -->
  </div> <!-- 모달 다이얼로그 -->
</div> <!-- 모달 전체 윈도우 -->
      
    
    -모달 사이즈 조절  (modal2.html)
     <div class="modal-dialog modal-lg">   => 큰사이즈
     <div class="modal-dialog modal-sm">  => 작은사이즈

    -링크를 클릭하지 않고, 페이지가 열림과 동시에 띄울 수 있다.  (modal-method.html)
 
     <script>
      $(function () {
       $('#myModal').modal({
          keyboard: false    //true를 주면 esc키를 누르면 모달이 닫힌다.
        })
      });
     </script>
        
     <div class="modal fade" id="myModal">

  
  - 클릭시 토글처리  메소드
     $('#myModal').modal('toggle');
     $('#myModal').modal('show');
     $('#myModal').modal('hide');

   <script>
        $(".btn").popover();         
   </script>

   <button type="button" class="btn btn-default" title="팝오버 제목" data-container="body" data-toggle="popover" data-placement="left" data-content="여기는 팝오버 부분에 대한 내용이 들어갑니다.">

    </button>

11. Affix
    
   -화면 스크롤시 특정 메뉴가 고정되는 효과
   -고정 시키려는 요소에  data-spy="affix" 값을 지정한다.
    data-offset-top="상단의 스크롤의 거리" ,  data-offset-bottom="하단의 스크롤의 거리"

    <ul class="nav nav-tabs nav-stacked" data-spy="affix" data-offset-top="140">
                <li class="active"><a href="#one">Section One</a></li>
                <li><a href="#two">Section Two</a></li>
                <li><a href="#three">Section Three</a></li>
